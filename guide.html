Guide on how to create your first local Repository and link it with Github

  1. Firstly you need to sign up on Github
  2. After this click on the Create new Repository in menu bar.
  3. After filling out the name and decription of repositiory click the Create new Repository Button
  4. Download and install Git from the link https://git-scm.com/download/win
  5. Create a Folder and in the Folder Right click and Click Git Bash Here Option
  6. Git bash terminal will open then write this command and enter git init

    init Command initiate a local in the current folder.
  7. To Check the Status of Git use this command git status

    At this stage the files are in unstaged position.
  8. Add these files in staged area using the command git add .
  9. To Commit the changes use the command git commit -m "message"
  10. If you make any changes than status changes and the changed files resides in unstaging area.
  11. To make files ready to add on unstaging area use this command git add index.html
  12. Now again to add the files to staging area use the previous command.
  13. Commit the changes and saved them.
  14. To link the remote Repository with your computer's local Repository use this command git remote add origin url.
  15. To push the branch master's work to origin remote Repository use this command git push -u origin master
  16. To see that files are uploaded to web repositiory go to Git hub on Chrome and refresh the page.

    Files are uploaded with track of time.
